### Technologies
- Ruby on Rails
- Rspec
- Bootstrap
- PostgresSQL database
- Cloudinary API
- Sass


### Features
- Creating a profile
- Creating pet profiles
- Uploading pet pictures
- Following other users
- Personal Calendar
- Activity-feed with other users


### Gems
- gem 'rspec-rails'
- gem 'factory_girl_rails'
- gem 'ffaker'
- gem 'masonry-rails'
- gem 'bootstrap-sass'
- gem 'carrierwave'
- gem "acts_as_follower"
- gem 'public_activity'
- gem 'will_paginate'


### Planned Features
- Integration of Petfinder API for adoptable pets
- Messenger
- Search
- Comments

### Setup
- Clone/Download this repo
- Run 'bundle install'
- Run 'rake db:create'
- Run 'rake db:migrate'
- Run 'rails server'
